Tool sử dụng trong dự án:

1. Go (Golang)
   - Ngôn ngữ lập trình chính
   - Sử dụng goroutine để quét song song
   - Yêu cầu: Go 1.21+

2. Masscan
   - Quét cổng IP tốc độ cao
   - Output JSON để Go parse
   - Yêu cầu quyền root/Administrator
   - Cài đặt:
     + Linux:   sudo apt install masscan
     + macOS:   brew install masscan
     + Windows: choco install masscan hoặc tải từ GitHub

3. MariaDB 10.11+
   - Lưu trữ dữ liệu IP, host, dịch vụ
   - Driver Go: github.com/go-sql-driver/mysql
   - Cài đặt:
     + Linux:   sudo apt install mariadb-server
     + macOS:   brew install mariadb
     + Windows: choco install mariadb hoặc tải từ mariadb.org
4. Nmap 7.9+
   - Quét cổng chi tiết (Port Scan) ở chế độ Stealth (-sS)
   - Yêu cầu: Nmap cài đặt trên hệ thống
   - Cài đặt:
     + Linux:   sudo apt install nmap
     + macOS:   brew install nmap
     + Windows: Tải từ nmap.org

Kiểm tra và Tự động Cài đặt:
   Chạy lệnh: ./scan_ip check
   → Tool sẽ tự động kiểm tra: masscan, nmap, mysql, mysqldump.
   → Đặc biệt trên Linux: Nếu thiếu tool, hệ thống sẽ hỏi và hỗ trợ TỰ ĐỘNG CÀI ĐẶT cho bạn.
